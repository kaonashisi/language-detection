Spoken language API Documentation

Endpoint 1: /languages-infos

This endpoint retrieves the total number of person speaking a language as a first and second language and total number of person speaking it in millions.

GET

Retrieve the id, language, the l1, l2 and total number of speakers for all languages

Endpoint 2: /languages-infos/?include_details={include_details}

This endpoint include details about the languages, their families and branches when the parameter include_details is 1 (by default it's 0)

GET

Retrieve the id, language, family and branch for all the languagesc

Endpoint 3: /languages-infos/{info_id}

This endpoint retrieves all the information about a specific language thanks to the info_id.

GET

Retrieve all information about language

Endpoint 4: /languages-audios

This endpoint retrieves all the audio links available.

GET

Retrieve links and languages name

REMARKS

To listen to the audio file, you can click on the link, download it and listen to it

Endpoint 5: /languages-audios?language_code={language_code}

This endpoint retrieves a list of audio files for a specific language according to the language_code.

GET

Retrieve language name and list of audio files

REMARKS

To listen to the audio file, you can click on the link, download it and listen to it

The only language you can request are English (en), German (de) and Spanish (es)
